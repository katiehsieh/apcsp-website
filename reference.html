Linux Reference Page

Command Description Example
man [command] access built in manuals for commands and file format man ssh-keygen
mkdir [name] makes [name] directory mkdir website
rmdir [name] deletes [name] directory rmdir website
cd [directory] goes to directory home; if directory is not specified, goes back to default home cd scripts
nano [file] opens [file], creates if it doesn’t exist, and edits if it exists nano myfirstbashscript
cp [file] [destination] copy [file] into [destination] cp myfirstbashscript scripts
mv [file] [destination] move [file] into [destination] mv index.html website
rm [file] deletes [file] rm mysecondbashscript
ping [ip address] sends packets to destination ip address to check if ping is successful ping rpi06
reboot shuts down and restarts Linux system reboot
sudo [command] “super user do” or “switch user do” allows user with permissions to perform [command] as that user sudo pip3 install network
ls lets user preview files within directory ls
chmod +x [filename] add executable permission to file chmod +x rpistats.cgi
